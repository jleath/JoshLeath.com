I am a software engineer with multiple years of experience with Ruby, JavaScript, Go and various frameworks.

I recently built Chimera, an open-source tool for automating canary deployments of containerized microservices.

Chimera is an open-source tool for performing automated canary deployments of containerized microservices, allowing software development teams to safely and easily deploy new versions of their code by taking advantage of the features provided by their existing service mesh.

Chimera is built with Node.js and is made available as a containerized web application with a React front end user interface, and an express app backend that handles managing AWS resources.
